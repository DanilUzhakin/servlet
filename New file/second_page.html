Изучение веб-технологий

HTML CSS JavaScript

HTML

  • HTML — это язык разметки для создания веб-страниц.
  • Используется для структурирования контента.
  • Состоит из тегов.

CSS

  • CSS — это язык стилей для оформления веб-страниц.
  • Позволяет задавать цвета, шрифты, отступы и другие параметры.
  • CSS бывает встроенным, внутренним и внешним.

JavaScript

  • JavaScript — это язык программирования для добавления интерактивности.
  • Позволяет создавать динамические страницы.
  • Используется в браузерах и на стороне сервера (Node.js).
